# WatchNext

WatchNext is a responsive movie and television discovery platform designed
to help users search, explore and organise entertainment content.

## Project Overview

The platform allows users to discover movies and television shows, view
content information, search by title and maintain a personal watchlist.

## Main Features

- Movie and television-show discovery
- Search functionality
- Detailed movie and show information
- Watchlist management
- Recently viewed content
- User registration and login
- Responsive design for desktop and mobile devices
- Privacy policy, terms and settings pages
